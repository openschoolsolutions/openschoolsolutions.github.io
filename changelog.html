Welcome to our changelog page

We are constantly trying to improve our platform and we want to be as transparent as possible about it.

March 1st, 2021 - First private beta update: 0.2.5

New features

  • We now can have active and non active users. After registering, every user needs activation by the admin of the platform.
  • We rewrote our progress indicators for showing the Topic progress of each student.
  • Now you can paste images in your text editor and they are automatically uploaded to the server.
  • You are now able to publish and unpublish all the content of a Competency right from its view page.

Bug fixes & improvements

  • Duplicating competencies is now much faster and reliable.
  • We improved the UX for multiple questions Assessments.
  • Now we have much better peformance when exporting Competencies. Now the exported files are much readable.
  • We extended the Automated tests across all aspects of the platform so we ensure that all functionality is always properly tested.
  • Now all users should use the Username field to mark their identities. This change was made to ensure the safety and privacy of all our users.

January 1st, 2021 - First private beta: 0.2.0

Bug fixes & improvements

  • Revamped UI elements to improve readability and semantics.
  • Backend improvements to make the functionality much faster and reliable.

September 24th, 2020 - First private alpha

New features

  • Now we can create content with HTML structure thanks to the addition of a CKEditor to all pages where is needed.
  • When there is no a Super Admin account, we can create one by accessing the install route.
  • There is more automated testing that will be expanded for the next version.
  • We can now enabled and disable subjects.
  • We have now a profile page where we can update our data and credentials.

Bug fixes & improvements

  • When creating a new Assessment for a Topic, we no longer have data loss if there is some validation error.
  • Now when there is no title in the page, there will not be a separator at top.
