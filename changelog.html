Welcome to our changelog page

We are constantly trying to improve our platform and we want to be as transparent as possible about it.

September 24th, 2020 - Super Admin and CKEditor

New features

  • Now we can create content with HTML structure thanks to the addition of a CKEditor to all pages where is needed.
  • When there is no a Super Admin account, we can create one by accessing the install route.
  • There is more automated testing that will be expanded for the next version.
  • We can now enabled and disable subjects.
  • We have now a profile page where we can update our data and credentials.

Bug fixes & improvements

  • When creating a new Assessment for a Topic, we no longer have data loss if there is some validation error.
  • Now when there is no title in the page, there will not be a separator at top.

September 8th, 2020 - Import/Export

New features

  • Export competencies with all their assessments, topics, learning paths and sections. Import the competencies to a Subject easily.
  • Import students from Excel files.
  • Track student progress by storing the answers from Assessments and marking LPs as completed.

Bug fixes & improvements

  • Improve the layout of all pages by creating a Title, Pretitle blocks.
  • Improve the layout of all pages by defining a more flexible approach with row and column layouts.
  • Updated OG tags to have a more consistent URL thumbnail when sharing pages from the platform.
  • UI improvements for the mobile and desktop layouts. Review of the Sidebar and Topbar for more consistent behavior.
